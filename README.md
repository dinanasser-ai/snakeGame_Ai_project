Snake Game AI Project
This is an Artificial Intelligence-based Snake Game project developed during my 3rd year of university as part of my academic coursework.

Project Overview
The project demonstrates the implementation of AI algorithms to enable the snake to navigate the environment and collect food efficiently. It was built to apply theoretical concepts of search algorithms and logic in a practical environment.

Key Features
AI Integration: Uses intelligent pathfinding to play the game.

Academic Project: Developed during the junior year of my Computer Science/Software Engineering studies.

Technologies: Built using Python
